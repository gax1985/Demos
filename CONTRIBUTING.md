# Contributing to TheAgora

Thank you for your interest in contributing!  
This project exists to serve communities, and your help makes it stronger.

## Ways to Contribute
- Reporting bugs
- Suggesting features
- Improving documentation
- Writing scrapers for new regions
- Enhancing sentiment analysis
- Improving security

## Development Setup
1. Install JDK 17+
2. Install Android Studio or IntelliJ IDEA
3. Clone the repository
4. Run `./gradlew build`

## Pull Requests
- Fork the repository
- Create a feature branch
- Write clear commit messages
- Ensure code is formatted and tested
- Submit a PR using the provided template

## Code Style
We follow Kotlin official style guidelines.
